智聊 · 静态原型

对话聊天 + 对话生图 Web 应用 · 移动端优先 · 包含完整的用户端与运营后台。

用户端 · Client

新聊天 (空状态)
居中欢迎语 + 圆角输入条 + 输入条内选择对话模型。
对话进行中
用户气泡 / AI 流式回复 / 代码块 / 操作工具栏 (复制、赞、踩、重新生成)。
生成图片
与对话页同结构 · 输入提示词 · 输入条内选择图片模型。
我的
个人信息 + 积分卡 + 数据概览 + 偏好设置 + 退出确认弹框。
积分流水
收入 / 支出 切换,按日期分组,展示充值与消耗记录。
登录
手机/邮箱 + 密码 + 验证码登录 + 第三方登录入口。
注册
完整的注册表单,带验证码与服务条款勾选。

运营后台 · Admin

管理员登录
账号 + 密码 + 2FA 验证码。
仪表盘 含图表
4 项核心数据 + 趋势折线图 + 模型占比环图 + 最近活动 + 系统健康。
AI 对话模型
卡片网格,支持启用/禁用切换 + 编辑弹框配置 endpoint、Key、价格、人设。
生图模型
文生图 / 图生图 / 图像编辑 多类型管理。
用户管理
列表 + 角色(普通/Plus)筛选 + 调整积分 + 变更会员级别弹框。
用户详情
基本资料 + 积分趋势 + 多 Tab 数据(对话/生图/积分/登录)。
生成记录 含详情抽屉
点击任意行右侧弹出抽屉:完整 Prompt / AI 回复 / 失败原因 / 上游链路。
积分流水
充值 / 消耗 / 奖励 / 管理调整 / 退款 全类型。
系统日志
控制台风格,按 INFO / WARN / ERROR / DEBUG 着色,支持实时跟随。
