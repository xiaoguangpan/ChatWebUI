性能与稳定性 · 验收方法

1. 启动耗时:使用 Android Vitals 或 Xcode Instruments 多次冷启动取 P95,目标 ≤ 2000ms。

2. 滚动帧率:用 GfxInfo / FPS Monitor 录制 30 秒长列表滚动,掉帧率 < 5%。

3. 内存:启动到聊天列表稳态,使用 Memory Profiler 观察 5 分钟,无明显增长。

4. Crash:接入 Bugly / Sentry,灰度 7 天 Crash-Free Sessions ≥ 99.9%。

5. 弱网:用 Network Link Conditioner 模拟 3G、丢包 5%,核心功能可用。
